# rabota-dla-koda-buguchego
Консольное Python‑приложение для учёта прочитанных книг.

## Функционал
* Добавление книг с оценкой и датой прочтения
* Просмотр всех книг
* Средняя оценка прочитанных книг
* Статистика по авторам
* Удаление книг

## Как запустить
1. Убедитесь, что установлен Python 3.x
2. Скачайте репозиторий
3. Запустите: `python main.py`
